DRESSIFY PRO - AI Outfit Website
=================================

SETUP:

1. Backend:
   cd server
   npm install
   node server.js

2. Frontend:
   cd client
   npm install
   npm run dev

IMPORTANT:
Replace YOUR_HF_API_KEY in server/server.js with your HuggingFace API key.
Get your free key at: https://huggingface.co/settings/tokens

HOW TO USE:
- Open http://localhost:5173 in browser
- Click "Try AI Outfit" on home page
- Upload your photo
- Enter a prompt like "casual summer outfit" or "formal ethnic wear"
- Click Generate
